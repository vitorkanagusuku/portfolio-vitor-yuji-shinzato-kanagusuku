# 🧼 Projeto Zaestravita - Site Institucional

## 📝 Descrição do Projeto
O Projeto Zaestravita consiste no desenvolvimento de um site institucional para uma empresa fictícia do ramo de produtos de limpeza pesada.

O objetivo do site é apresentar a empresa, seus produtos e facilitar o contato com clientes, utilizando uma interface moderna, responsiva e de fácil navegação.

Este projeto foi desenvolvido como parte das atividades acadêmicas, aplicando conceitos de desenvolvimento web.

---

## 🎯 Objetivo
Criar um site funcional e visualmente agradável para representar uma empresa, destacando seus produtos e melhorando sua presença digital.

---

## 🚀 Tecnologias Utilizadas

- **HTML5** — Estrutura do site  
- **CSS3** — Estilização e layout responsivo  
- **JavaScript** — Interatividade e manipulação do DOM  

---

## 📊 Funcionalidades

- Menu de navegação com versão mobile (hambúrguer)  
- Navegação por seções (Home, Produtos, Sobre, Contato)  
- Seção de destaque (hero) com chamada visual  
- Exibição de produtos com imagens  
- Layout responsivo (adaptado para celular)  
- Formulário de contato com validação básica  
- Animações simples para melhor experiência do usuário
