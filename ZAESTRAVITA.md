# 🧼 Projeto Zaestravita - Site Institucional

## 📝 Descrição do Projeto
O Projeto Zaestravita consiste no desenvolvimento de um site institucional voltado para a divulgação de produtos de limpeza pesada.

O site foi criado com base em uma necessidade real: apresentar os produtos de uma empresa de forma organizada, clara e acessível para possíveis clientes. O projeto foi desenvolvido em grupo como parte das atividades acadêmicas, aplicando conceitos de desenvolvimento web.

---

## 🎯 Objetivo
Criar uma plataforma simples e funcional para exibir produtos, facilitar a navegação e melhorar a apresentação da marca no ambiente digital.

---

## 🚀 Tecnologias Utilizadas

* **HTML5** — Estruturação das páginas  
* **CSS3** — Estilização e layout  
* **Ferramentas:** VS Code  

---

## 📊 Funcionalidades

- Página inicial com apresentação da empresa  
- Exibição de produtos de limpeza  
- Organização visual das informações  
- Layout responsivo básico  
- Navegação entre páginas  

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram adquiridos conhecimentos importantes como:

- Estruturação de páginas com HTML
